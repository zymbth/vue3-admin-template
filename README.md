# Vue Admin Template

## Via yarn

### Project setup
```bash
yarn install
```

### Compiles and hot-reloads for development
```bash
yarn dev
```

### Compiles and minifies for production
```bash
yarn build
```
## Via npm

### delete yarn file
delete 'yarn.lock'

### Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run dev
```

### Compiles and minifies for production
```bash
npm run build
```
