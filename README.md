# Vue3 Admin Template

在大佬的基础上[PanJiaChen / vue-admin-template](https://github.com/PanJiaChen/vue-admin-template)，更换为vue3 & element plus，方便个人后续使用

[线上地址](http://ymzhao.work/vue3-admin-template)

## 安装依赖

```bash
yarn install
# or
npm install
```

## 开发

```bash
yarn dev
# or
npm run dev
```

## 打包

```bash
yarn build
# or
npm run build
```
